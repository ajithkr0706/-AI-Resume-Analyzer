# -AI-Resume-Analyzer

A web app built with **Python, Flask, and NLP (spaCy)** to analyze resumes.  
It extracts skills, checks keyword match percentage, and shows missing skills.

## ⚙️ Tech Stack
- **Frontend:** HTML, CSS, JS  
- **Backend:** Python (Flask)  
- **AI/NLP:** spaCy  
- **PDF Reading:** PyMuPDF
